# Prototype Design

## Page Map
- Inbox Page
- Ticket Detail Page
- Assignment Panel
- Note Composer

## Interaction Rules
- The inbox defaults to newest tickets first and shows visible SLA risk.
- Clicking a row opens the ticket detail without losing current filters.
- Assignment and status changes require explicit user action and show success or failure feedback.
- Internal notes are separated from customer-visible conversation data.
