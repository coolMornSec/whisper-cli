# 任务工作区模板

## 目录约定
- 每个任务都必须放在 `tasks/<task-id>/` 下。
- 阶段产物必须存放在固定的 `00` 到 `09` 目录中。
- `state.json` 记录运行时执行状态，`manifest.json` 记录静态任务策略。
- `AGENT.md` 定义所有参与 Agent 的任务级执行约定。

## 核心交付物
- `AGENT.md`
- `01-spec/spec.md`
- `04-design/prototype.md`
- `04-design/api-contract.yaml`
- `05-rules/rules.md`
- `06-tests/test-cases.md`
- `09-audit/review.md`
