# Acceptance Criteria

## Acceptance Checklist
- Inbox list renders tickets with subject, requester, priority, status, owner, and SLA badge.
- Filters update the list without a full page reload.
- Ticket detail shows conversation summary, internal notes, and assignment controls.
- Assignment and status changes persist through the API and appear immediately in the UI.
- Internal notes are saved with author and timestamp.
- Workflow validation and case tests pass.
